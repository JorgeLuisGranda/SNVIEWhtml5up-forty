Css

CSS es uno de los lenguajes base de la Open Web y posee una especificación estandarizada por parte del W3C. Desarrollado en niveles, CSS1 es ahora obsoleto, CSS2.1 es una recomendación y CSS3, ahora dividido en módulos más pequeños, está progresando en camino al estándar.

El Elemento Css <width>(ancho del elemento).

El Elemento Css <background-color>, el color de fondo del contenido y del relleno.

El Elemento Css <color>, el color del contenido del elemento (generalmente texto).

El Elemento Css <text-shadow>: coloca una sombra difuminada en el texto dentro del elemento.

El Elemento Css <display>: selecciona el modo de visualización para el elemento.
