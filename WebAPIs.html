Web APIs

Éstas pueden ser accesadas mediante código JavaScript, y te permiten hacer cualquier cosa, desde realizar ajustes menores a cualquier window ó element, hasta generar efectos gráficos y de aúdio complejos usando APIs tales como WebGL y Web Audio.

El Elemento de WebAPIs <Api de Dispositivos>Este conjunto de APIs permiten acceder a varias funciones de hardware disponibles para páginas Web y aplicaciones. Por ejemplo, API de Sensor Luz Ambiental, API de Estado de Batería, API de Geolocalización, API de Bloqueo de Puntero, API de Proximidad, API de Orientación de Dispositivo, API de Orientación de Pantalla, API de Vibración.

El Elemento de WebAPIs <Api de Comunicación>Estas APIs le permiten a páginas Web y aplicaciones comunicarse con otras páginas o dispositivos. Por ejemplo, API de Información de Red, Notificaciones Web, API de Push Simple.

El Elemento de WebAPIs <Api de gestión de datos>Los datos de los usuarios pueden ser almacenados y gestionados usando este conjunto de APIs. Por ejemplo, FileHandle API, IndexedDB.

El Elemento de WebAPIs <Api Privilegiadas>Una aplicación privilegiada es una aplicación instalada que ha recibido derechos específicos por parte del usuario. Entre las APIs privilegiadas se encuentran: TCP Socket API, API de Contactos, API de Almacenamiento de Dispositivos, API de Navegador,

El Elemento de WebAPIs <Api Certificada>Una aplicación certificada es una aplicación de bajo nivel que ejecuta operaciones críticas sobre un sistema operativo como Firefox OS. Las aplicaciones menos privilegiadas interactúan con estas aplicaciones certificadas mediante Actividades Web. Entre las APIs Certificadas se incluyen: API Bluetooth, API de Conexión Móvil, API de Estadísticas de Red, Telefonía, WebSMS, API de Información WíFi, API de Cámara, API de Administración de Energía, API de Ajustes, API de Inactividad, API de Permisos, API de Hora/Reloj.
